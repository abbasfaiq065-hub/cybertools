#🛡️ CYBER-TOOLS

Welcome to CYBER-TOOLS, a collection of Python-based cybersecurity and networking tools developed as part of my cybersecurity learning journey.

This repository contains command-line tools focused on networking, information gathering, password security, and web security. Each project is organized into its own folder with source code, documentation, and any required dependencies.

---

📂 Included Tools

🌐 IP Info

Retrieve detailed information about IPv4 and IPv6 addresses.

📡 IP Pinger

Test network connectivity and measure response times.

🔎 Port Scanner

Scan predefined TCP ports on a target host.

🛡️ Vulnerability Scanner

Perform basic web security assessments by testing for common vulnerabilities and security misconfigurations.

🔐 Password Generator

Generate strong and random passwords using Python.

✅ Password Strength Checker

Evaluate password strength and provide recommendations for improvement.

🌍 Subdomain Enumerator

Discover publicly accessible subdomains for a target domain.

📖 WHOIS Lookup

Retrieve publicly available WHOIS registration information for domain names.
